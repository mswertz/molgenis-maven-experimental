# EXPERIMENTAL MOLGENIS MODULES

This maven project contains some experimental module for the molgenis-maven project.
This code does not compile but contains useful bits to be merged in the normal molgenis-maven

* molgenis-core contains a JAXB parser for the molgenis.xml files. This is the first part useful to port over.
* molgenis-db contains (incomplete) database layer
* molgenis-ui contains (incomplete) ui layer
* molgenis-observ is an example application
